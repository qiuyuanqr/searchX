# 海光信息（688041.SH）· 13 周展望

## 一屏结论

未来 13 周方向：**高位震荡转跌**（基准情景约 50%）。

- **基本面**：2025 全年营收 143.77 亿元（+56.9%）、归母净利润 25.45 亿元（+31.8%），但低于此前券商 30–32 亿元一致预期，Q4 单季净利环比下滑；2026 一季报营收 +68.06%、净利 +35.82%，落在预告区间上半区，同期经营现金流同比骤降 97.32%。
- **估值（2026-07-01 收盘，akshare 核准）**：股价 349.90 元（单日 -5.51%）、总市值 8,132.86 亿元、动态 PE 295.91 倍、PB 35.13 倍——PE/PB 均处自身近三年 99.6% 分位，约为半导体行业中位数（99.63 倍）的 3 倍。
- **题材**：国产 CPU+DCU"双芯战略"叙事，但 DCU 市占率约 5%（媒体估算），显著落后华为昇腾（约 20%）；与 DeepSeek 等大模型的合作目前多为"适配"层面，未见可验证的批量商用订单。
- **关键事件**：(1) 2026 年 8 月上旬（推断）中报披露；(2) 2026-09-17~19 华为全联接大会；(3) 融资余额近两周连续下降，产业资本年内被净减持超 20 亿元。

**操作倾向（条件式）**：若已持有，按条件管理回撤；若尚未建仓，宜观望——不追高。整体置信度：中。

## 核心驱动 Top3 / 核心风险 Top3

**驱动**：(1) 一季报营收 +68.06%、净利 +35.82%，DCU 放量趋势明确；(2)"十五五"规划+美国出口管制收紧客观强化国产替代逻辑；(3) DeepSeek 系列持续官宣"Day 0 适配"，生态曝光度高。

## 业务结构

- CPU（海光通用处理器，兼容 x86）+ DCU（深算系列加速卡），分产品收入占比未拆分披露
- 前五大客户营收占比超 90%；第一大股东[[中科曙光]]持股 27.96%（2026Q1 末），历史上为重要客户
- 2016 年通过与 AMD 合资获得 x86 Zen1 架构永久授权，2019 年后 AMD 停止授权新架构，此后靠自主迭代
- 与中科曙光"世纪并购"（约 1,160 亿元换股）已于 2025-12-09 终止

## 竞争格局

- vs [[寒武纪]]（688256）：估值同处历史高位（PE 230 倍），营收增速/市占趋势更强，若情绪逆转更可能同步承压
- vs [[龙芯中科]]（688047）、[[景嘉微]]（300474）：仍处亏损/低增长，估值缺乏盈利支撑，弹性更多来自题材炒作
- DCU 国产市占约 5%，显著落后[[华为昇腾]]（约 20%，国产第一）

## 八条逻辑链（综合：↓4 / 纯震荡 1 / 偏涨或震荡偏涨 3，但支撑↓的更多基于已兑现迹象，支撑涨的多依赖尚未兑现条件）

1. 中报延续 60%+ 营收增速 → 高估值获阶段性消化支撑 → 震荡偏涨 / 中
2. 毛利率连续第四期下滑 → 坐实 DCU 放量稀释盈利质量的担忧 → ↓ / 中
3. Q1 经营现金流骤降 97% 若为趋势性因素 → 利润与现金创造能力分歧放大 → ↓ / 低
4. 产业资本中报窗口前后再现减持预披露 → 强化"越涨越卖"负反馈 → ↓ / 中
5. 半导体 ETF 溢价/杠杆风险进一步暴露 → 板块资金撤离叠加个股高波动 → ↓ / 中
6. 华为全联接大会释放信创生态积极信号 → 板块情绪阶段性催化 → 震荡偏涨 / 中
7. 美国 BIS 收紧对华 AI 芯片出口 → 需求侧替代逻辑与供给侧代工受限同步收紧，方向不完全一致 → 震荡 / 中
8. DCU 出现可验证批量商用订单（而非仅适配）→ 叙事从题材升级为业绩兑现 → ↑ / 低（当前无此证据）

## 三情景 + 验证信号

- **乐观（≈20%）**：中报延续 60%+ 增速且毛利率企稳、DCU 出现可验证批量商用案例。盯：中报毛利率是否止跌。
- **基准（≈50%）**：中报增速放缓至 30–40%、毛利率延续下滑、产业资本减持持续。盯：融资余额与主力资金能否止住净流出。
- **悲观（≈30%）**：中报大幅不及预期（重演 2025 年报剧本）、ETF/杠杆风险集中兑现。盯：上交所公告栏目新减持计划或问询函。

## 条件式策略

- **持有**：中报毛利率延续下滑、营收增速跌破 40% → 分批减仓。
- **持有**：融资余额连续两周以上净流出、主力资金连续 3 日净流出 → 视为退潮信号。
- **试仓**：仅在"中报毛利率企稳 + 可验证批量商用订单 + ETF 溢价回归正常"三者同时满足时小仓位试探，不追高。
- **退出**：新减持预披露 / 监管问询函 / 中报净利润环比同比双降 → 止损离场。

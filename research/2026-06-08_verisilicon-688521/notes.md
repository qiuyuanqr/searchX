# 芯原股份（688521.SH）· 13 周走势判断与触发条件

> 信息截止日期：2026-06-08（北京时间）。本笔记是 [report.html](./report.html) 的精简版，原始三件套见 `archive` 字段。不给目标价、不给买卖评级；操作一律条件化。

## 一屏结论（A）

未来 ~13 周方向判断：**震荡偏中性**，上下空间均不对称地大。

- **核心驱动 Top3**：① AI 算力订单爆发（1.1-4.29 新签 82.40 亿 / AI 占 91.37%）；② 双并购补全 IP 矩阵（芯来 RISC-V CPU + 逐点视觉 / 投影主控）；③ NPU IP 端侧 AI 渗透加速（91 家客户、140+ 芯片、2 亿颗出货）。
- **核心风险 Top3**：① 估值显著透支（PS LTM 36×，2026E 19×）；② 持续亏损 + 经营现金流持续负 + 资产负债率近 60%；③ 商誉跳升（1.78 → 6.29 亿）+ 大股东减持。
- **一句话操作倾向**：若尚未建仓，等估值消化或中报兑现再评估；若已持有，可按个人风险承受能力分批兑现部分浮盈、保留底仓博中报与 [[WAIC]] 催化，不追高。
- **整体置信度**：中（估值结论高，方向判断受短期事件扰动大）。
- **关键验证窗口**：① 2026-07-17 至 07-20 [[WAIC]] 2026 上海；② 2026-08-30 前 [[中报披露]]。

## 操作触发条件（L · 条件式）

- **试仓（未建仓者）**：月度订单连续两月 ≥20 亿 + 股价回落至近 90 日中位数附近 + 换手率 < 3% → 小仓位试探。
- **加仓（已有底仓）**：中报 H1 营收同比 ≥80% + 综合毛利率 ≥32% + 无显著商誉减值 → 业绩公告后 1-3 个交易日确认情绪后加仓。
- **止盈（条件式）**：周换手率连续两周 ≥5% + 月度订单环比首次明显放缓 → 分批兑现部分浮盈、保留底仓。
- **止损（条件式）**：中报单季营收增速 ≤50% 或综合毛利率跌破 30% 或公告大额商誉减值 → 按个人风险承受能力减仓 / 退出。

## 重点指标 Top3

1. 月度新签订单（自愿性披露）：20 亿/月 阈值。
2. 2026 H1 综合毛利率：32% 企稳意味着量产业务规模效应初现。

# 晋拓股份 603211.SH · 深度调研

> 「压铸老兵 + 机器人题材」组合，但<strong>实控人 5 个月套现近 3 亿 + Q1 业绩证伪 + IPO 项目达产严重不及预期</strong> 三大硬伤同框；当前 PE 静态 143×、PB 7.86× 已严重透支机器人题材。

## 核心结论（A 节）

未来 13 周（至 2026-09-22）方向 **震荡偏弱（基准 50% / 乐观 25% / 悲观 25%）**——A 股压铸"小盘 + 机器人题材"的典型组合，协作机器人客户含 UR / 越疆 / 节卡 / 艾利特 / 法奥，公司明示 2026 机器人占比目标 25%。但基本面 + 治理 + 估值三大空头逻辑硬于题材弹性 + 板块催化两大多头逻辑。

- **核心驱动 Top3**：① 协作机器人零部件先发 + 公司明示占比目标 25%；② 5 亿元定增预案（2026-04-29 落地，待审）；③ 13 周内机器人板块密集催化（WAIC 7 月上海、WRC 8 月北京、Tesla Optimus Gen3 量产口径 + Q2 财报）。
- **核心风险 Top3**：① 实控人方 2025-12 → 2026-05 累计减持 1200 万股套现 2.77 亿元（第一轮 734 万股 + 第二轮 466 万股），媒体口径"第三轮拟再套现 1 亿"；治理瑕疵（2025/11 换董秘 + 2026/3 证代离职未补聘）；② 2026Q1 营收 +12.09% / 归母 −35.68% / 扣非 −7.69% + IPO 募投项目结项 20 个月仍处产能爬坡（承诺达产年增 5.12 亿、累计仅 3118 万）；③ PE 静态 ≈ 143× / PB ≈ 7.86× / PS ≈ 7.68× 已为同业（旭升 / 爱柯迪 / 文灿）的 5-7 倍，主业毛利率仅 17.08%。

## 公司速览（B 节）

- 主营：铝合金精密压铸件（[[机器人]]结构件 + 汽车零部件 + 智能家居 + 信息传输设备 5G/卫星通讯）。
- 上市：2022-07-25 上交所主板。
- 控股：江苏智拓投资 47.48%（控股股东）；实控人张东 + 何文英（夫妻）通过江苏智拓 + 直接持股 + 员工持股平台合计控制约 70%+（精确合计比例媒体口径与一手公告 PDF 对账未本轮独立核成功，置信度中）。
- 基准日 2026-06-24 收盘 **36.22 元 / −2.11%**（akshare 新浪日线 sh603211 前复权核准）。
- 总股本约 2.72 亿股，**总市值 ≈ 98.5 亿元**。
- **静态 PE ≈ 143× / PB ≈ 7.86× / PS ≈ 7.68×**——A 股压铸板块上轨、同业 5-7 倍。
- 近 90 日（3/1 - 6/24）最高 37.80（基准日盘中）/ 最低 20.52 / **区间振幅 84.21%** / 平均换手率 3.87%——3 月底起一路走高、5 月底 6 月初 30% 回撤、6/18 因机器人题材涨停（+9.99%）后高位震荡。

## 行业地位（D 节）

- **规模**：A 股压铸第二梯队（同业旭升 50+ 亿 / 爱柯迪 60+ 亿 / 文灿 60+ 亿，晋拓仅 12.83 亿）。
- **差异化**：协作机器人结构件先发（UR / 越疆 / 节卡 / 艾利特 / 法奥）——窄护城河、易被切入。
- **同业切入**：爱柯迪并购卓尔博切微特电机、旭升集团抢人形机器人轻量化、文灿一体化压铸服务整车厂——**晋拓的"先发优势"有效期可能短于 12 个月**。

## 决策框架（L 节·条件式）

- **试仓**：股价回踩 28 元附近（自基准回撤 ≥22%）+ 业绩预告口径"扣非同比 ≥ −10%" + 试仓窗口内实控人未再披露减持，三者同时满足 → 小仓位试仓。
- **加仓**：半年报机器人单品收入首次单列 ≥ 1.5 亿 / 毛利率回升 ≥ 18% / 主力 10 日累计净流入 > 5000 万 中任两项满足 → 加仓。
- **减仓**：半年报扣非同比 < −20% / 实控人再次披露减持（第三轮拟 1 亿口径已公开）/ 毛利率破 16% / 主力 10 日累计净流出 > 1 亿 / 定增方案被否或显著延迟 中任一触发即考虑。

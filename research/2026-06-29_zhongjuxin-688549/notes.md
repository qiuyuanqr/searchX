# 中巨芯（688549.SH）· 13 周展望

## 一屏结论

未来 13 周方向：**偏跌 / 高位震荡转跌**（基准情景约 50%）。2026Q1 营收 +35.3% 扭亏，但 PE-TTM 近 2200 倍显著高于同业，六氟化钨题材公司已澄清「无大额订单、无扩产计划」，9/8 另有 IPO 三周年大额解禁；未建仓宜观望不追高。整体置信度：中。

- **基本面**：2025 全年营收 12.12 亿 +17.7%、归母净亏 1,659.62 万；2026 Q1 营收 +35.3% 扭亏；但 Q1 经营现金流 -1.71 亿（同比恶化）。
- **估值（2026-06-29 收盘，akshare 收盘后核准）**：股价 37.88 元（单日 +18.01%）、总市值 561 亿、PE-TTM 2,199.68 倍、PB 18.64 倍——显著高于同业可比（江化微 PE 147 / 雅克 92 / 安集 82 / 南大 124，剔除同属虚高的晶瑞 817）。
- **题材**：六氟化钨海外断供 + 国产替代催化，但公司 6/21 已明确"无大额订单、合资生产、无扩产计划"。
- **关键事件**：(1) 2026-08 下旬中报披露；(2) 2026-09-08 IPO 三周年限售解禁（有限售流通股占 82.29%）。

**操作倾向（条件式）**：若已持有，按条件管理回撤；若尚未建仓，宜观望——不追高。整体置信度：中。

## 核心驱动 Top3 / 核心风险 Top3

**驱动**：(1) 2026 Q1 营收 +35% 扭亏；(2) 六氟化钨涨价 + 国产替代主题催化；(3) 5/16 公告 8 亿元定增预案 + 沧州华北基地扩产。

**风险**：(1) PE-TTM 约 2,200 倍、PB 18.6，估值脱离基本面；(2) 公司亲口澄清无六氟化钨大额订单 + 合资生产 + 无扩产；(3) 9/8 IPO 三周年限售解禁（有限售流通股 82.29%）。

## 八条逻辑链（综合：↓4 / 震荡 2 / ↑2）

1. 六氟化钨海外断供 + 公司放量 → 业绩弹性 → 但公司亲口承认无大额订单 → 震荡 / 高
2. 湿化学品下游需求持续 → Q2/Q3 营收增速 → ↑ / 中
3. 商誉减值已出清 → 2026 全年净利转正 → 但 PE 约 2,200 仍贵 → 震荡 / 中
4. 定增过会 + 募投落地 → 短期 EPS 摊薄 → ↓（短期）/ 中
5. 9/8 IPO 三周年限售解禁 → 流通股结构变化、卖压 → ↓ / 中高
6. 上交所发问询函或监管工作函 → 短期题材冷却 → ↓ / 中
7. 板块情绪降温 → 高位筹码松动 → 震荡偏跌 / 高
8. 国产替代政策再加码或外部管制升级 → 板块联动重启 → ↑ / 中

## 三情景 + 验证信号

- **乐观（≈20%）**：六氟化钨大额订单兑现 + 中报营收 +50% + 板块二次拉升 → 突破 6/29 高点。盯：互动易 / 公告披露订单。
- **基准（≈50%）**：题材回落 + 限售解禁提前定价 + 中报营收 +30%。盯：换手率从 >20% 收敛至 <10% 即可能见顶。
- **悲观（≈30%）**：监管问询 + 控股股东减持 + 板块普跌。盯：上交所监管动态 + 减持预披露。

## 条件式策略
